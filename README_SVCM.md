# Protocolo SVCM (Scientific Version Control Manager)

## Sistema de Versionado Semántico Científico (SemVer-S)

- **MAJOR (X.0.0)**: Cambio de paradigma o validación formal de un postulado central.
- **MINOR (0.X.0)**: Adición de nuevas pruebas, simulaciones exitosas o integración de nuevas ramas.
- **PATCH (0.0.X)**: Correcciones de estilo, ajustes en la redacción o refinamiento de isomorfismos.

## Protocolo de Transición de Estados

1.  **[CONJETURA]**: Idea inicial con base lógica pero sin prueba matemática o computacional.
2.  **[EN PRUEBA]**: Bajo simulación o análisis de consistencia.
3.  **[FORMALIZADO]**: Con estructura matemática y lógica sólida.
4.  **[POSTULADO VALIDADO]**: Tras haber pasar pruebas de irreducibilidad computacional y coherencia interna.

## Flujo de Trabajo

1.  Recepción de input (texto/simulación).
2.  Comparación con versión anterior (Diff).
3.  Actualización de Preprint + Changelog.
4.  Análisis de Recursividad (impacto en otros preprints).
